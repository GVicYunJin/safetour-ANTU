# 安途-SafeTour — 行程报备与轨迹安全监护系统
项目中文名：安途
项目英文名：SafeTour

## 一、项目整体介绍
- 安途SafeTour，是一套开源私有化部署的出行安全监护系统。
- 项目核心主打两大王牌功能：前台一键智能报备、后台无感分钟级实时安全监控。
- 专为家庭出行、学生外出、结伴游玩、团队外勤场景打造，解决外出安全报备繁琐、家长无法实时掌握出行动态的痛点。
- 所有数据完全私有、自主可控，无第三方云端中转，杜绝隐私泄露风险。
- 项目采用服务端 Web 管理 + 安卓客户端 APP 双端架构，支持 Docker 轻量化私有化部署，普通人也能快速搭建专属出行监护系统。

## 二、项目核心两大王牌功能
### 1、前台核心：一键智能报备（项目最大亮点）
- APP采用极简相机界面，操作零门槛。
- 用户只需单击一个按钮，即可全自动完成全套上报操作。
- 自动采集高精度融合定位、海拔、全套设备状态数据，同步拍摄前置、后置双摄像头现场画面。
- 所有数据实时上传私有服务端，同时自动推送报备信息与现场照片至 QQ 群。
- 家长和群内亲友可以实时查看外出人员的实时位置和现场实况，彻底替代人工打字报备，出行报备高效、直观、安心。

### 2、后台核心：无感分钟级实时安全监控
- APP 获取权限后可长期后台静默保活运行。
- 即使锁屏、退出界面、手动清理后台，监控服务也不会中断。
- 系统默认每分钟自动上报一次全套设备与位置数据。
- 包含高精度融合 GPS 位置、定位精度、海拔、加速度、传感器数据、气压、设备温度、手机电量、基站信息等全维度数据。
- 服务端可实时查看设备在线状态、实时定位，同时支持完整历史行程轨迹回溯回放，实现全天候不间断出行安全监护。

## 三、项目技术架构
### 1、服务端
- 后端基于 Python Flask 框架开发
- 前端使用 HTML、CSS、JavaScript、Vue 开发
- 支持 Docker compose一键部署，可部署在服务器、NAS、支持 Docker 的路由器等所有设备，资源占用极低、适配性极强

### 2、客户端
- 基于 Android Studio 开发
- 适配大部分安卓设备，后台保活能力强，运行稳定

## 四、服务端功能介绍
服务端分为管理员后台、普通用户前台，权限完全隔离，安全规范。

### 1、管理员功能
- 支持用户账号管理、注册管理、权限分级分发
- 支持成员管理、出行任务管理
- 实时可视化查看所有终端设备在线状态、精准位置
- 全方位监测设备电量、海拔、传感器、气压、设备温度、基站等硬件数据
- 支持完整历史行程轨迹回放、行程溯源
- 实时查看所有用户的一键报备双摄照片与报备记录
- 内置 NapTech QQ 推送接口，自动推送报备动态至指定 QQ 群

### 2、普通用户功能
- 通过管理员分配的专属密码登录
- 可查看个人出行轨迹、历史报备记录与个人设备状态

## 五、安卓客户端完整功能
### 1、全自动无感后台上报
- 授权后常驻后台静默运行，不受锁屏、退后台、清后台影响
- 一分钟周期自动上传全套定位与传感数据，24 小时不间断监控

### 2、一键双摄智能报备
- 单键触发，同步完成前后置双摄拍照、位置采集、设备数据上报
- 自动推送 QQ 群，紧急场景可快速完成报备、实况同步

## 六、项目核心特色
1. 双模式核心设计，兼顾即时报备和全天候监控
2. 全程私有化部署，数据自留、隐私绝对可控
3. Docker 轻量部署，家用设备即可搭建
4. 安卓强力后台保活，杜绝失联、漏报问题
5. QQ 群自动联动推送，适配家庭监护场景

## 七、适用场景
- 中小学生独自外出、上下学安全监护
- 家庭亲子出游、多人结伴旅行实时报备
- 户外徒步、露营、远行安全追踪记录
- 团队外勤、小型出行团队统一行程管理
- 个人日常出行轨迹自主备份、安全留存
